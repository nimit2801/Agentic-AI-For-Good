# Abstract Illustration Style Guide — Agentic AI For Good

## Overview
Warm earth-tone editorial illustrations for blogs, LinkedIn posts, and marketing materials.
Generated via Canva at **1080x1080** (Instagram post format) for maximum versatility.

## Base Prompt Template
```
Abstract warm-toned illustration of [TOPIC DESCRIPTION]. Style: flat illustration with warm earth tones - beige (#F5F1EB), terracotta/burnt orange (#D4754E), warm browns, soft blacks (#1A1A1A), muted taupe (#6B6560). [SCENE-SPECIFIC DETAILS]. NO text, NO words, NO logos, NO photographs. Think editorial magazine illustration style.
```

## Canva Settings
- **Design type:** `instagram_post` (1080x1080)
- **Format:** PNG export
- **Note:** Canva sometimes adds text overlays despite "NO text" in prompt. May need manual removal via editing transaction.

## Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| Warm Beige | `#F5F1EB` | Backgrounds, light areas |
| Terracotta | `#D4754E` | Accent, warm highlights |
| Near Black | `#1A1A1A` | Dark elements, contrast |
| Muted Taupe | `#6B6560` | Secondary tones, shadows |
| Warm Browns | various | Mid-tones, skin, earth |

## Style Keywords
- Flat illustration
- Warm earth tones
- Editorial magazine style
- Minimalist
- Abstract silhouettes
- Geometric shapes
- NO stock photos, NO AI-techy imagery, NO gradients

## How to Create More
1. Copy the **Base Prompt Template** above
2. Replace `[TOPIC DESCRIPTION]` with your new topic
3. Replace `[SCENE-SPECIFIC DETAILS]` with 1-2 sentences describing the visual scene
4. Generate in Canva using `generate-design` with `design_type: instagram_post`
5. Save best candidate with `create-design-from-candidate`
6. Export as PNG with `export-design`
7. Download to `public/images/` and review in browser
